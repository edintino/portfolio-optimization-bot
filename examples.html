Example Runs and Plots

Example 1: Maximizing Return

Parameters:

  • Investment Day: 2022-01-01
  • Optimization Metric: Maximize Return
  • Portfolio Optimization Interval: 30 days
  • Optimization History: 365 days
  • Top N Stocks: 25
  • Ticker Update Interval: 90 days
  • Initial Capital: $1,000
  • Transaction Cost Rate: 0.001

Results:

  • Final Optimized Portfolio Value: $2,679.22
  • Total Return (Optimized): 167.92%
  • Final Uniform Portfolio Value: $1,945.78
  • Total Return (Uniform): 94.58%

Plot:

*The plot shows that the optimized portfolio (blue line) outperformed the uniform portfolio (orange line) over the investment period.*
